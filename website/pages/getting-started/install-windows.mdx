# Claude Code Installation Guide for Windows

> **Time:** ~10 to 20 minutes | **Prerequisites:** Windows 10 or later, permission to install apps

Most users only need the **Quick Start** below. The advanced sections are only for corporate VPN/proxy issues or managed setups.

---

## Quick Start

### Step 1: Install Git for Windows

1. Go to **https://git-scm.com/download/win**
2. Download **Git for Windows**
3. Run the installer with the default options

### Step 2: Open PowerShell

Press `Windows`, type `PowerShell`, and open it.

### Step 3: Install Claude Code

Run this command:

```powershell
irm https://claude.ai/install.ps1 | iex
```

### Step 4: If Claude cannot find Git Bash, run this once

Some Windows setups need one extra copy-paste step:

```powershell
$bashPath = @(
  "$env:LOCALAPPDATA\Programs\Git\bin\bash.exe",
  "C:\Program Files\Git\bin\bash.exe"
) | Where-Object { Test-Path $_ } | Select-Object -First 1

if ($bashPath) {
  [Environment]::SetEnvironmentVariable("CLAUDE_CODE_GIT_BASH_PATH", $bashPath, "User")
  Write-Host "Saved Git Bash path:" $bashPath
} else {
  Write-Host "Git Bash not found. Install Git for Windows first."
}
```

### Step 5: Close PowerShell, reopen it, and start Claude Code

```powershell
claude
```

Claude Code should open and ask you to sign in in your browser. After login, return to PowerShell and continue in the terminal window.

### Step 6: Confirm it works

```powershell
claude --version
```

If you see a version number and `claude` opens, you are done.

<details>
<summary>Advanced: install with npm instead</summary>

Use this only if the Windows installer is blocked or does not work.

1. Install **Node.js LTS** from **https://nodejs.org/**
2. Open a fresh PowerShell window
3. Run:

```powershell
npm install -g @anthropic-ai/claude-code
```

</details>

<details>
<summary>Advanced: corporate VPN or proxy certificate setup</summary>

Only use this if Claude Code installs but cannot connect because of SSL or certificate errors.

```powershell
New-Item -ItemType Directory -Force -Path C:\tools

$allCerts = @()
$allCerts += Get-ChildItem Cert:\CurrentUser\Root
$allCerts += Get-ChildItem Cert:\CurrentUser\CA

$pemContent = ""
foreach ($cert in $allCerts) {
  $pemContent += "-----BEGIN CERTIFICATE-----`n"
  $pemContent += [System.Convert]::ToBase64String($cert.RawData, [System.Base64FormattingOptions]::InsertLineBreaks)
  $pemContent += "`n-----END CERTIFICATE-----`n`n"
}

$pemContent | Out-File -FilePath "C:\tools\corporate-certs.pem" -Encoding ASCII
[Environment]::SetEnvironmentVariable("NODE_EXTRA_CA_CERTS", "C:\tools\corporate-certs.pem", "User")
```

Close and reopen PowerShell, then try `claude` again.

</details>

## Troubleshooting

### `claude` command not recognized

1. Close all PowerShell windows
2. Open a fresh one
3. Run `claude --version`
4. If it still fails, rerun the installer or use the npm option above

### Claude says Git Bash is missing

```powershell
$env:CLAUDE_CODE_GIT_BASH_PATH
```

If nothing prints, rerun the Git Bash setup step in the Quick Start section.

### SSL or certificate errors

Use the **corporate VPN or proxy certificate setup** section above.

### Still stuck

Use the simpler cross-platform guide at [Install Claude Code](/getting-started/install), or ask your facilitator or IT support to complete the advanced setup with you.
