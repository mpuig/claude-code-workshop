# Claude Code Workshop

Practical training for consultancy teams on using Claude Code to build solutions without writing code.

## Who Is This For?

Non-technical insurance professionals who want to use AI coding agents to accelerate their work: data analysis, process prototyping, reporting automation, and rapid application building.

## Fictional Company: Mediterranean Insurance Group (MIG)

All materials use a consistent fictional context: **Mediterranean Insurance Group** -- a mid-size European insurer headquartered in Barcelona, operating across Spain, Portugal, Italy, and Southern France.

See [MIG Context](/company-context/company) for the full company profile, product lines, personas, and market landscape.

## Suggested Training Flow

| Order | Module | Duration | Description |
|-------|--------|----------|-------------|
| 1 | [Module 0 - Getting Started](/getting-started) | 10 min | Install Claude Code, authenticate, first session |
| 2 | [Module 1.1 - First Steps](/fundamentals/first-steps) | 30 min | Natural language instructions, file operations, 3 scenarios |
| 3 | [Module 1.2 - Project Memory](/fundamentals/project-memory) | 15 min | CLAUDE.md for consistent insurance context |
| 4 | [Module 2.1 - Underwriting Brief](/insurance-workflows/underwriting-brief) | 30 min | Commercial property underwriting workflow |
| 5 | [Module 2.2 - Loss Ratio Analysis](/insurance-workflows/loss-ratio-analysis) | 30 min | Motor portfolio analysis with visualizations |
| 6 | [Module 2.3 - Market Assessment](/insurance-workflows/market-assessment) | 30 min | Portugal market entry via web research |
| 7 | [Module 3 - Vibe Coding](/vibe-coding) | 45 min | Build a Claims Dashboard app |

**Total guided time: ~3 hours** (modules can be done independently)

## Hands-On Exercises

These are standalone exercises designed for the live session. Each includes step-by-step prompts, expected outputs, and rich fake data.

| Exercise | Persona | Duration | What You Build |
|----------|---------|----------|----------------|
| [1 - Data Insight Tool](/exercises/data-insight-tool) | Isabel Santos (Strategy) | 40 min | Market white space analysis with interactive charts |
| [2 - Claims Triage](/exercises/claims-triage) | Ana Martinez (Claims) | 45 min | Claims classification, prioritization & routing prototype |
| [3 - Automated Reporting](/exercises/automated-reporting) | Pierre Dupont (Actuarial) | 55 min | Quarterly claims report with auto-update capability |

## Prerequisites

- A Claude account with Claude Code access
- A terminal (macOS Terminal, Windows PowerShell, or WSL)
- No programming knowledge required

## Quick Start

```bash
# 1. Install Claude Code
curl -fsSL https://claude.ai/install.sh | bash

# 2. Navigate to an exercise
cd exercises/exercise-1-data-insight-tool

# 3. Start Claude Code
claude

# 4. Follow the prompts in the README
```
