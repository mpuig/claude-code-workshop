# Module 3: Practice Exercises

These are **independent, hands-on challenges** designed for the live workshop session. Each exercise gives you a business problem, a realistic persona, and the files or context you need -- you figure out how to solve it with Claude Code.

Unlike the [Guided Workflows](/insurance-workflows) in Module 2, these exercises do not walk you through every prompt. They tell you what to achieve and give you tips, but the approach is yours to decide. This is where you put what you have learned into practice.

---

## The Exercises

| Exercise | Duration | What You Build |
|----------|----------|----------------|
| [1: Executive Onramp](/exercises/executive-onramp) | 90 min | Zero-to-one executive prompting workflow with copy/paste wins and progressive challenge |
| [2: Data Insight Tool](/exercises/data-insight-tool) | 40 min | Market white space analysis across 68 regions with interactive charts and an executive memo |
| [3: Claims Triage](/exercises/claims-triage) | 45 min | Classification, prioritization, and routing system for 220 incoming claims |
| [4: Automated Reporting](/exercises/automated-reporting) | 55 min | Quarterly claims report with visualizations and a reusable auto-update script |

Each exercise includes the files it needs. Exercises 2 to 4 use downloadable data files, and Exercise 1 uses the shared MIG context files. See the [Data Catalog](/exercises/data-catalog) for a full reference to the exercise datasets.

---

## How to Start

1. For Exercise 1, open your terminal in the repository root. For Exercises 2 to 4, open it in the exercise folder you want to run.
2. Start Claude Code with `claude`
3. Follow the instructions in the exercise page

---

## Tips

- **Start by exploring the data.** Ask Claude to read and summarize the CSV files before jumping into analysis.
- **Be specific about output format.** If you want a table, chart, or memo, say so explicitly.
- **Iterate.** The first output is rarely the final one. Refine with follow-up prompts.
- **If you get stuck**, check the troubleshooting section at the bottom of each exercise page.
